PASTA DE FOTOS
==============

Coloque suas fotos aqui nesta pasta.

Formatos aceitos:
- JPG / JPEG
- PNG
- WEBP
- GIF

Como referenciar no script.js:
  foto: "fotos/nome-da-foto.jpg"

Dicas:
- Use nomes sem espaços (use hifens: primeira-viagem.jpg)
- Otimize as imagens para web (max 1MB por foto)
- Recomendado: 1200x900px ou maior
