PASTA DE ÍCONES PWA
===================

Coloque os ícones do app aqui.

Tamanhos necessários:
- icon-48.png
- icon-72.png
- icon-96.png
- icon-128.png
- icon-144.png
- icon-152.png
- icon-180.png
- icon-192.png
- icon-384.png
- icon-512.png

Dica: Use https://www.pwabuilder.com/imageGenerator para gerar
todos os tamanhos a partir de uma imagem 512x512.
